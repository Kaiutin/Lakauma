About

Authors: Samuel Uusi-Mäkelä, Anton Laurila, Jari-Matti Kankaanpää ja Kristiina Manninen

... is an application to help visualize the locations of rental apartments in Jyväskylä region. The application should present a Google maps map, with different search-matching apartments marked. Clicking a marker will open a small infowindow for that apartment, which contains basic information about the apartment and a link to the original housing agency that owns the apartment.

For more information,see the project page.
